COLOR GUESSING GAME!

Personal Extra Project

This is an extra game I personally create. As its name, this is a gussing game where player will guess the location of a particular color.

There are 50 boxes. All of the box's color will be hidden. However, the color of the boxes will be the same except for 1 box.

Player's task is to find this 1 special box among other boxes.

To make it more interesting, player only has 10 attempt to give it a guess.

One last thing, the boxes will be shuffled on each round. Good Luck!
